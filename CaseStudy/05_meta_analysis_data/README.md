# MetaFlow Analysis Inputs and Outputs

This folder contains zipped MetaFlow Analyzer input files and output results for the radiotherapy and anti-tumor immunity case study.

## Purpose

MetaFlow Analyzer is the statistical analysis and visualization module of MacAma. In this case study, quantitative chart data were manually extracted from included studies, organized into standardized input files, uploaded to MetaFlow Analyzer, and used to generate statistical outputs and visualizations.

Each zip file in this folder corresponds to one immune-related outcome and contains the original analysis input data and the output figures or result files generated by MetaFlow Analyzer.

## Contents of Each Zip File

Each zip file may include:

- the standardized input file uploaded to MetaFlow Analyzer;
- exported statistical result files;
- forest plots;
- subgroup analysis outputs, where available;
- funnel plots, where available;

## Analysis Performed

In the current case study, MetaFlow Analyzer used standardized mean difference (SMD) as the primary effect measure for continuous immune-related outcomes. Depending on the uploaded data and user-selected parameters, the platform generated study-level SMD estimates, pooled SMD estimates, 95% confidence intervals, heterogeneity statistics, subgroup analysis outputs, and funnel plots for visual assessment of potential reporting bias.
